## 📝 Project Overview
This project demonstrates a scalable **Smart Inventory & Sales Manager**. It follows a **Vertical Inheritance** model, showing how a foundational logic script evolves into an AI-driven enterprise solution.

### 🎨 Visual Standards
- **Industry Theme:** Industrial Asphalt (`#34495E`) & Alert Orange (`#E67E22`)
- **Typography:** Segoe UI / System Sans-Serif

---

## 🚀 The 4-Tier Evolution

### 🟢 Level 01: Beginner (Foundational Logic)
- **Language:** Python
- **Focus:** Data structures, CLI interaction, and core arithmetic logic.
- **Key Feature:** Pure algorithm for stock valuation.

### 🔵 Level 02: Intermediate (The Interface)
- **Language:** HTML5, CSS3, JavaScript
- **Focus:** Re-implementing Level 01 into a Visual Dashboard.
- **Key Feature:** Persistent storage using Browser LocalStorage and Responsive Design.

### 🌑 Level 03: Professional (Enterprise System)
- **Language:** PHP, MySQL, React
- **Focus:** High-concurrency data management and server-side security.
- **Key Feature:** Relational Database integration and Prepared SQL Statements.

### 🟣 Level 04: Expert (The Intelligent Engine)
- **Language:** Python (AI), C++ (High Speed)
- **Focus:** Predictive Analytics and Performance Optimization.
- **Key Feature:** Linear Regression model to forecast stockouts based on Level 03 data.

---

## ⚙️ How to Run
1. **Beginner:** Run `python 01-Beginner/main.py`
2. **Intermediate:** Open `02-Intermediate/index.html` in any browser.
3. **Professional:** Host `03-Professional/` on a PHP server (XAMPP/WAMP) and import the `.sql` file.
4. **Expert:** Install requirements `pip install pandas scikit-learn` and run the forecasting script.
